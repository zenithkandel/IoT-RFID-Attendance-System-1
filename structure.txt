================================================================================
                        PROJECT OVERVIEW
================================================================================

EDU TRACK PRO is a production-ready IoT-based smart attendance management 
system that uses RFID (Radio Frequency Identification) technology to automate
the process of tracking student attendance. The system eliminates manual
attendance marking, reduces errors, and provides real-time data analytics.

                        USER ROLES & FEATURES
================================================================================

ADMINISTRATOR (Admin Panel)

================================================================================
                     DATABASE ARCHITECTURE
================================================================================

DATABASE: edutrack (MySQL/MariaDB)
Backend: XAMPP Server (PHP + MySQL)

TABLE 1: students
─────────────────
  • ID (INT, Primary Key, Auto Increment) - Unique student identifier
  • UID (TEXT) - RFID card unique identifier
  • Name (TEXT) - Student full name
  • Roll (INT) - Roll number (used for login)
  • Class (TEXT) - Student's class/grade
  • Address (INT) - Student address
  • password (TEXT) - Hashed password for student login
  • last_login (TEXT) - Last login timestamp

TABLE 2: attendance
───────────────────
  • ID (INT, Primary Key, Auto Increment) - Unique attendance record ID
  • Time (INT) - Attendance timestamp (Unix time)
  • UID (INT) - Reference to student UID
  • Date (TEXT) - Attendance date (formatted)
  • Type (INT) - Attendance type (0=Absent, 1=Present, etc.)

TABLE 3: users
──────────────
  • ID (INT, Primary Key, Auto Increment) - Unique admin user ID
  • username (TEXT) - Admin username
  • password (TEXT) - Hashed password
  • last_login (TEXT) - Last login timestamp
